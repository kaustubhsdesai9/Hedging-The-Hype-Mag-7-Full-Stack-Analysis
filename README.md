# Hedging the Hype: A Full-Stack Macro Analysis of the Magnificent 7

## Executive Summary
This project delivers a high-impact, full-stack quantitative analysis of the "Magnificent 7" (AAPL, AMZN, GOOGL, META, MSFT, NVDA, TSLA) within the context of global macro-economic shifts. By analyzing performance across different VIX regimes, interest rate cycles, and yield sensitivity, this study provides a framework for "Hedging the Hype" while identifying sustainable risk-adjusted returns in an AI-driven market.

## Key Objectives
*   **Risk Mitigation:** Quantify drawdown profiles and rolling Sharpe ratios to evaluate macro-fragility.
*   **Macro-Nexus Analysis:** Correlate basket performance with yield curve movements and volatility regimes.
*   **Visual Storytelling:** Deploy interactive visualizations (HTML) to illustrate stock performance races and intra-basket correlations.

## Methodology
The analysis employs a **Multi-Stage Quantitative Pipeline**:
1.  **Data Ingestion:** Aggregation of historical price data and macro-economic indicators (Yields, VIX).
2.  **Statistical Processing:** Calculation of rolling betas, correlations, and drawdown metrics.
3.  **Regime Analysis:** Segmentation of returns based on VIX levels and "Era" specific correlation heatmaps.
4.  **Interactive Sharing:** Conversion of complex data into Plotly/Altair-based interactive HTML charts.

## Tools & Technologies
*   **Analytics:** Python (Pandas, NumPy, Scipy).
*   **Visualization:** Plotly, Matplotlib, Seaborn.
*   **Macro Data:** Federal Reserve Economic Data (FRED), Yahoo Finance.
*   **Reporting:** Executive Summary (PDF) and Interactive HTML Dashboards.

## Key Insights (Preview)
*   **Stock Performance Race:** Visual tracking of capital concentration in AI leaders.
*   **VIX Regime Returns:** Comparative analysis of how Mag 7 performs during high-volatility "Fear" markets.
*   **Yield Sensitivity:** Identifying which tech giants are most exposed to "Higher for Longer" rate environments.
